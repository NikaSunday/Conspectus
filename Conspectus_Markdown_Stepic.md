# Markdown. Конспект курса (Stepic)

## Базовый синтаксис

### Заголовки:

Заголовки имеют 6 уровней. Чем "главнее" заголовок, тем ниже его уровень. Заголовок отмечается соответствующим кол-вом знаков октоторпа (решетки) в начале строки (после решетки обязателен пробел).

Альтернативный способ для заголовков - 

Заголовок 1 уровна
========================

Заголовок уровнем пожиже
------------------------

### Параграфы:

Для создания параграфа используется пустая строка.
Перед параграфом не стоит добавлять табуляции или пробелы. Для отступов можно использовать код символа &nsbp; (Non-breaking space).

#### В Markdown нет концепции выравнивания текста (кроме таблиц).

Однако для этого можно использовать HTML-тег \<center\>
````
<center>Этот текст по центру<center>
````

А вот так оно выглядит:
<center>Этот текст по центру</center>

### Переносы строк

Для простого переноса строки (без создания нового абзаца) в конце строки нужно вставить 2 пробела перед переводом  
(здесь тестово переведена строка) строки.

Можно использовать тег HTML\<br\>:
````
Вышел немец<br>Из тумана...
````

**А вот так оно выглядит:**
Вышел немец<br>Из тумана...

### Акценты в тексте:
#### Жирный текст:

Две звездочки или два подчеркивания вокруг текста:
````
**bold text - 2 asterisk**
__bold text - 2 underline__
````

**А вот так оно выглядит:**
**bold text - 2 asterisk**
__bold text - 2 underline__

#### Курсив

Одна звездочка или одна подчеркивание вокруг текста:
````
*italic text - astrisk*
_italic text - underline_
````

**А вот так оно выглядит:**
*italic text - astrisk*
_italic text - underline_

#### Жирный курсив:
Три звездочки или одна подчеркивание вокруг текста:
````
***bold italic text - 3 astrisk***
___bold italic text - 3 underline___
````

**А вот так оно выглядит:**
***bold italic text - 3 astrisk***
___bold italic text - 3 underline___

#### Зачеркнутый текст:
две тильды \~\~ вокруг текста:
````
~~Strikethrough text - 2 tilda~~
````

**А вот так оно выглядит:**
~~Strikethrough text - 2 tilda~~

#### Цветной текст

Штатно Markdown **не поддерживает** цаетной текст, но если процессор Markdown поддерживает HTML, можно использовать тег \<font\> с атрибутом ````color````, используя шестнадцатеричный код #RRGGBB^
````
<font color="red">Красный-красный текст!</font>
````

**А вот так оно выглядит:**
<font color="red">Красный-красный текст!</font>

### Блок цитат

#### Простой блок цитат:
Поставить стрелку вправо ````>```` перед текстом:
````
> Тут цитата
````

**А вот так оно выглядит:**
> Тут цитата
>
> #### Блок цитат с несколькими абзацами:
>Перед каждым абзацем ставится  ````>````:
````
> Строка цитаты 1 **со** _всякою_ ~~разметкою~~
> Строка цитаты 2
````

**А вот так оно выглядит:**
> Строка цитаты 1 **со** _всякою_ ~~разметкою~~
> Строка цитаты 2

> #### Вложенные блоки цитат:
>Перед каждым абзацем ставится  ````>```` в нужном количестве:
````
> Строка цитаты 1 **со** _всякою_ ~~разметкою~~
>> Строка цитаты 2 **со** _всякою_ ~~разметкою~~
>>> Строка цитаты 3 **со** _всякою_ ~~разметкою~~
````

**А вот так оно выглядит:**
> Строка цитаты 1 **со** _всякою_ ~~разметкою~~
>> Строка цитаты 2 **со** _всякою_ ~~разметкою~~
>>> Строка цитаты 3 **со** _всякою_ ~~разметкою~~

### Списки (Lists):

#### Неупорядоченные списки (Unordered lists):

Создаются с использованием символов ```` -, * или +````

**А вот так оно выглядит:**
- пункт раз с \-
- пункт два
  
* пункт раз с \*
* пункт два
  
+ пункт раз с \+
+ пункт два

#### Упорядоченные списки (Ordered lists):
Создаются с использованием числовых значений и точек:
````
1. Пункт раз
2. Пункт два
3. Ну и куда деваться, пункт три
````

**А вот так оно выглядит:**
1. Пункт раз
2. Пункт два
3. Ну и куда деваться, пункт три


#### В  списки можно добавлять другие списки, блоки цитат, кода, изображения и текст.

### Код

Для вставки кода используются обратные кавычки \`\`\`\`

**А вот так оно выглядит:**
````
тут код
````
В тексте можно вставлять короткие фрагменты кода, обрамляя их одной парой кавычек: \`тут код\`

**А вот так оно выглядит:**
Это текст `а тут код` и тут текст


Чтобы подсветить синтаксис, нужно подставить имя языка после трех обратных кавычек:

````
```javascript
function logMessage() {
  console.log("Сообщение в консоль");
}
```
````

**А вот так оно выглядит:**
```javascript
function logMessage() {
  console.log("Сообщение в консоль");
}
```

### Горизонтальные линии (Horizontal Rules):

Для создания горизотальных линий используются вынесенные на отдельную строку три дефиса `---`, три подчеркивания `___` или три звездочки `***` (можно использовать и бОльшее количество символов, главное, чтобы не меньше трех):

````
Глава первая: Исчезновение Уилла Байерса

---

Глава вторая: Чудак с Кленовой улицы

***

Глава третья: Здорово, Джолли

___
````
**А вот так оно выглядит:**
Глава первая: Исчезновение Уилла Байерса

---

Глава вторая: Чудак с Кленовой улицы

***

Глава третья: Здорово, Джолли

___


### Ссылки (Links) для перехода к другим ресурсам или документам:

С помощью квадратных и круглых скобок:
````
[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things)
````

**А вот так оно выглядит:**
[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things)


#### Ссылки с заголовками (Title):
т.е. со всплывающей при наведении курсора подсказкой:
````
[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things "Американский научно-фантастический драматический сериал ужасов, созданный братьями Даффер для стриминг-сервиса Netflix")
````

**А вот так оно выглядит:**
[Очень странные дела](https://en.wikipedia.org/wiki/Stranger_Things "Американский научно-фантастический драматический сериал ужасов, созданный братьями Даффер для стриминг-сервиса Netflix")

**Ссылки можно форматировать стандартными методами Markdown:
````
Мне нравится **[Stranger _Things_]**(https://en.wikipedia.org/wiki/Stranger_Things)**.
````

**А вот так оно выглядит:**
Мне нравится **[Stranger _Things_](https://en.wikipedia.org/wiki/Stranger_Things)**.

Чтобы быстро создать ссылку на URL или адрес электронной почты, заключите их в угловые скобки:
````
<https://en.wikipedia.org/wiki/Stranger_Things>
<eleven@strangerthings.com>
````

**А вот так оно выглядит:**
<https://en.wikipedia.org/wiki/Stranger_Things>
<eleven@strangerthings.com>


### Изображения (Images):

Для вставки изображений используется следующий синтаксис:
````
![Добро пожаловать в Хоукинс](https://ik.imagekit.io/awilum/welcome-to-hawkins.jpg)
````

**А вот так оно выглядит:**
![Добро пожаловать в Хоукинс](https://ik.imagekit.io/awilum/welcome-to-hawkins.jpg)

Если изображение локальное, то используется *относительный путь*

**Чтобы изображение распознавалось как ссылка**, ссылка на изображение заключается в квадратные скобки, после которых приводится url в курглых скобках:
````
[![Добро пожаловать в Хоукинс](https://ik.imagekit.io/awilum/welcome-to-hawkins.jpg)](https://en.wikipedia.org/wiki/Stranger_Things)
````

**А вот так оно выглядит:**
[![Добро пожаловать в Хоукинс](https://ik.imagekit.io/awilum/welcome-to-hawkins.jpg)](https://en.wikipedia.org/wiki/Stranger_Things)

Для изменения размера изображения необходимо, чтобы процессор поддерживал HTML, в таком случае используется HTML-тег `img` с атрибутами `width` и `height`:
````
<img alt="Добро пожаловать в Хоукинс" height="100" src="https://ik.imagekit.io/awilum/welcome-to-hawkins.jpg" width="150" />
````
(тут не поддерживает)


### Экранирующие символы (Escaping Characters):

Если в тексте нужно отобразить спецсимвол, который используется в Markdown для форматирования текста, необходимо использовать обратный слеш `\`:
````
\- Очень \*странные\* дела
````

**А вот так оно выглядит:**
\- Очень \*странные\* дела

Список символов, нуждающихся в экранировании:
\\ 	        обратный слеш (backslash)
\` 	        обратная кавычка (backtick)
\* 	        звездочка (asterisk)
\_ 	        подчеркивание (underscore)
\{ \} 	    фигурные скобки (curly braces)
\[ \] 	    квадратные скобки (brackets)
\< \> 	    угловые скобки (angle brackets)
\( \) 	    круглые скобки (parentheses)
\# 	        знак номера (pound sign)
\+ 	        плюс (plus sign)
\- 	        минус (minus sign, hyphen)
\. 	        точка (dot)
\! 	        восклицательный знак (exclamation mark)
\| 	        вертикальная черта (pipe)

### Создание таблицы:

Для создания таблицы используются вертикальные черты ```` | ```` для разделения ячеек и тире (минус) ```` - ````  для разделения заголовков и строк таблицы.
